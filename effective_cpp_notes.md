### 5. **明确编译器为你创建了哪些函数**
当我们定义一个空类的时候，编译器会默默为我们提供五个必须的函数:
- 构造函数
- 拷贝构造函数
- 析构函数
- 复制赋值函数
> 需要注意的是，c++11之后编译器其实会为我们默认提供一个**移动构造函数**，**移动复制函数**
```c++
class Demo{
  // 空类
  // 编译器为我们提供：
  Demo(){} // ---> 默认构造函数
  Demo(const Demo& other){/* 调用所有类成员的赋值函数 */} //---> 默认拷贝函数
  Demo(Demo&&); // 
  Demo& operator=(const Demo& other);
  ~Demo(){} // --->默认析构函数
};
```
